Moving Average

Menghitung rata-rata dari data historis tahunan untuk memprediksi nilai masa depan. Cocok untuk trend jangka panjang.

Linear Regression

Menggunakan metode statistik untuk menemukan hubungan linear antara waktu dan penjualan. Cocok untuk data dengan tren naik/turun konsisten.

Trend Analysis

Menggabungkan analisis tren dengan pola musiman untuk prediksi yang lebih akurat. Cocok untuk data dengan pola berulang (seasonal).
